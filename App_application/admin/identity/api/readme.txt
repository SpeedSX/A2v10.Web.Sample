Как это работает
================

1. В таблице a2security.Users добавлено новое поле ApiUser.
Такой пользователь не может войти в систему. У него нет мейла/телефона.
Он не показывается в общем списке пользователей.

2. Добавлена дополнительная таблица a2security.ApiSettings. Там может быть несколько записей
